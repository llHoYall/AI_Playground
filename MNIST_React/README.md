# MNIST React

Handwriting number recognition with svm and react.

## Usage

1. $ node 1-download.js
2. $ node 2-preprocess.js
3. $ node 3-batch.js
4. $ node 4-train.js
5. $ node 5-test.js
6. $ npm run compile
7. $ node 6-server.js
8. open chrome with `http://localhost:3002`

## Requirements

You must to use the node.js with version @8.

- node@8
